# Online Grocery Store 🛒

An online grocery shopping platform built with PHP, MySQL, HTML, CSS, and JavaScript.

## Features
- Browse products by category
- Search for products by keywords
- View product details and availability
- Add products to a shopping cart
- Edit and manage the shopping cart
- Place orders and receive confirmation
- View purchase history (user)
- Admin dashboard for viewing all purchases

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## Setup
1. Clone the repository
2. Set up a local server (e.g., MAMP)
3. Import `products.sql` to your MySQL server
4. Access the website via localhost
